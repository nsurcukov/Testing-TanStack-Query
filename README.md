# Лист с бесконечным скроллом и виртуализацией

## Описание проекта

Простое веб-приложение для управления задачами с акцентом на высокую производительность при большом количестве данных:

- **CRUD-операции** (создание, чтение, обновление, удаление задач)
- **Бесконечный скролл** — задачи подгружаются с сервера порциями по мере прокрутки
- **Виртуализация списка** — рендерится только видимая часть
- Данные хранятся на бэкенде (подходит для json-server, mock API или реального сервера)

## Используемые технологии

- **React** + **Vite**
- **TypeScript**
- **TanStack Query**  — управление запросами, кэширование, infinite scroll
- **TanStack Virtual**  — виртуализация списка
- **Fetch API** — взаимодействие с сервером
- Стили: обычный SCSS

## Функциональность

- Просмотр списка задач 
- Добавление новой задачи
- Редактирование существующей задачи
- Удаление задачи
- Автоматическая подгрузка задач при скролле вниз
- Индикатор загрузки при получении новых порций
- Адаптивный и быстрый интерфейс даже при большом объёме данных
 
 ## Запустить проект  

 ```npm run start:prod```
